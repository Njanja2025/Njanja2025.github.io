This module outlines the key endpoints and configurations for integrating Njax AI services.

1. AI Trigger Endpoint

URL: https://api.njaxmarket.net/trigger

Usage: Activate prompt-based workflows or deploy AI bots via POST requests.

2. API Key Access

To get an API key, create a developer account and navigate to your Njax Dashboard.

Header format: Authorization: Bearer YOUR_API_KEY

3. Supported AI Models

  • Njax GPT (custom)
  • OpenAI (GPT-4/ChatGPT)
  • Claude AI
  • Gemini Pro
  • Midjourney (via webhook)

4. Response Format

All endpoints return JSON by default.

{
  "status": "success",
  "data": {
    "response": "AI result here..."
  }
}
